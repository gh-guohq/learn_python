# 变态跳台阶

## c

```
class Solution {
public:
    int jumpFloorII(int number) {
       if (number == 0)
           return 1;
       int sum = 0;
       for(int i=0;i<number;i++)
            sum += jumpFloorII(i);
        return sum;
    }
};
```

## python
```
# -*- coding:utf-8 -*-
class Solution:
    def jumpFloorII(self, number):
        # write code here
        if number == 0 :
            return 1
        sum = 0
        for i in range(number):
            sum += self.jumpFloorII(i)
         
        return sum
```
